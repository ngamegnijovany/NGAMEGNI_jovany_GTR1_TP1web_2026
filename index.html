Introduction

Ce mini-site a été créé dans le cadre du cours de TP programmation web.

Il permet de découvrir la creation de plusieurs pages HTML reliées entre elles.

Objectif du mini-site

L'objectif est d'apprendre à structurer un petit project web avec plusiers fichiers.

Pages disponibles

  • La page Accueil présente le contexte générale du mini-site.
  • La page Présentation contient les informations perssonnelles des l'étudiant.
  • La page contact contient les information des contat de l'étudiant.
